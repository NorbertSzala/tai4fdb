5. MVP workflow

Goal: import one organism and calculate basic sequence-level statistics.

Steps:

1. Create database schema.
2. Import organism metadata.
3. Import assembly metadata.
4. Register input files.
5. Import CDS FASTA.
6. Validate CDS sequences.
7. Calculate sequence metrics:
   - length
   - GC
   - GC3
   - ambiguous bases
   - internal stop codons
8. Calculate codon usage per CDS.
9. Import tRNAscan-SE output.
10. Calculate tRNA copy number per anticodon.
11. Calculate codon weights.
12. Calculate tAI per CDS.
13. Store all results in the database.
14. Export summary tables.
15. Generate first plots.


6. Quality control

CDS-level QC:

- length divisible by 3
- no internal stop codons
- valid start codon
- valid terminal stop codon
- no ambiguous nucleotides
- minimum length threshold
- duplicated sequence IDs
- duplicated gene IDs
- extreme GC content
- extreme GC3 content

Assembly-level QC:

- number of CDS
- number of valid CDS
- number of excluded CDS
- total CDS length
- mean CDS length
- median CDS length
- number of predicted tRNA genes
- number of unique anticodons
- missing amino acid families
- unusual codon usage patterns


7. Codon usage

Store codon counts in long format:

- cds_id
- codon
- count

Useful summaries:

- codon counts per CDS
- codon counts per assembly
- codon frequency per assembly
- relative synonymous codon usage
- codon usage comparison between organisms

Important filters:

- exclude stop codons
- exclude codons containing ambiguous bases
- optionally exclude very short CDS
- respect organism-specific genetic code

9. tAI calculation

Required inputs:

- CDS codon counts
- tRNA anticodon counts
- codon-to-anticodon pairing rules
- genetic code
- codon weights

Store:

- tAI value per CDS
- method name
- method version
- weight table
- analysis run ID
- excluded codons
- genetic code used

Validation:

- compare with stAIcalc
- compare with previous scripts
- compare with manually calculated toy examples
- calculate correlation between methods
- calculate mean absolute error
- inspect outliers
